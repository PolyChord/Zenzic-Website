Optimising the number of antennas

During our work with Warwick Manufacturing Group, we optimised the placement of the antennas on the Warwick campus to yield the best possible network coverage of the main road network to facilitate the deployment of connected and autonomous vehicles. Our results were verified via WMG's user-in-the-loop simulations, in which state-of-the-art ray tracing simulations are combined with emulation of a physical signal.

Use the slider to control the number of antennas in the network and see how the coverage changes.

Optimising for hardware failure

Network hardware is bound to fail at some point, therefore it is important (and essential in safety-critical applications) for the remaining transmitters to maintain acceptable coverage of the cityscape. MIDAS can optimise sensor locations with antenna failure in mind - this guarantees a network that is as robust to hardware failures as is possible under the installation constraints.

Use the slider to control which antenna is dropped from the network and see how the coverage changes.
